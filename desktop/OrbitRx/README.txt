OrbitRx Desktop Launchers
=========================

This folder is a template. On Windows, run ONE of these from the repo:

  1. Double-click "Setup Desktop.bat" in the repo root  (recommended)
  2. Or run scripts\setup_desktop.bat

That copies launchers to:  Desktop\OrbitRx\

Then double-click "Run OrbitRx.bat" on your Desktop to start the app.

Note: The cloud development environment cannot create folders on your
local Windows desktop — you must run Setup Desktop.bat once on your PC.
